    # 算法复杂度

    2019.03.07

    一段程序的好坏从两个指标上衡量，时间复杂复和空间复杂度。时间复杂度是指执行这个算法所需要的计算工作量；空间复杂度是指执行这个算法所需要的内存空间。这两个指标个人认为是在我们敲下代码之前必须要思考的两个出发点，想到了就能写得一手好代码！想不到，只能对产品经理呵呵呵。

    # 时间复杂度

    • 理论上讲，一个算法的执行时间我们是没办法通过公式或什么算出来的，但是我们也不可能去机器上去实测一下，对我们程序员来讲其实只要需要知道哪个算法执行花的时间多，哪个算法执行花的时间少就可以了，而且算法的执行时间和算法的执行次数肯定是正比的，所以哪个算法执行次数多，那么花费的时间肯定多。一个算法中的语句执行次数称为语句频度或时间频度。记为 T(n) ; n 称为问题的规模，当 n 不断变化时，时间频度 T(n) 也会不断变化。使用 O 记号法（大 O 记号）表示最坏运行情况的上界。

    • 大O记号推导方法：

    ① 用常数1取代运行时间中的所有加法常数。

    ② 在修改后的运行次数函数中，只保留最高阶项。

    ③ 如果最高阶项存在且不是1，则去除与这个项目相乘的常数。得到的结果就是大O阶。

    # 常数阶

    下面这个算法总共执行了三次，T(n) = 3 用 1 代替常数 3 时间复杂度为 0(1)。

    let sum = 0, n = 100;       //执行一次
sum = (1 + n) * n / 2;      //执行一次
console.log(sum);           //执行一次

    # 线性阶

    循环体执行n次，时间复杂度为 O(n)

    for(var i = 0; i < n; i++){
    //执行n次
}

    # 对数阶

    由2^x=n 得到x=log2^n 即 logn （log以2为底的对数函数为常用对数，省略底数2，即logn）,时间复杂度为 O(logn)。

    var count = 1;      
while (count < n){
    count = count * 2;
    // 执行 log2^n 次   
}

    # 平方阶

    循环体执行 n X n 次，时间复杂度为 O(n^2)

    for(var i = 0; i < n; i++){
    for(var j = 0; j < n; j++){
        // 执行 n X n 次  
    }
}

    # 立方阶

    循环体执行 n X n X n 次，时间复杂度为 O(n^3)

    for(var i = 0; i < n; i++){
    for(var j = 0; j < n; j++){
        for(var j = 0; j < n; j++){
            // 执行 n X n X n 次  
        }
    }
}

    算法执行次数的函数 大O阶 非正式术语

    3n2+2n+1 O(n2) 平方阶

    # 时间复杂度从小到大为（越小越好）：

    O(1)<O(logn)<O(n)<O(nlogn)<O(n2)<O(n3)<O(2n)<O(n!)<O(nn)

    # 空间复杂度

    百度百科读一遍：

    • 空间复杂度(Space Complexity)是对一个算法在运行过程中临时占用存储空间大小的量度，记做S(n)=O(f(n))。

    同样也是用大O来表示。

    • 如对于递归算法来说，一般都比较简短，算法本身所占用的存储空间较少，但运行时需要一个附加堆栈，从而占用较多的临时工作单元；若写成非递归算法，一般可能比较长，算法本身占用的存储空间较多，但运行时将可能需要较少的存储单元。

    • 计算方法：

    ①算法的空间不随被处理数据量n的大小而改变时，可表示为O(1)。（冒泡排序）

    ②算法的空间以2为底的n的对数成正比时，可表示为O(log2n)。（快速排序）

    ③算法的空间复杂度与n成线性比例关系时，可表示为O(n)。（一般的递归算法）

    ④递归算法的空间复杂度=递归深度N*每次递归所要的辅助空间。

    ⑤对于单线程来说，递归有运行时堆栈，求的是递归最深的那一次压栈所耗费的空间的个数，因为递归最深的那一次所耗费的空间足以容纳它所有递归过程。

    # 总结

    做为一名资深程序员，知道自己敲的代码是好是坏吗？时间复杂度和空间复杂度是算法的尺子和磅秤，只有身高和体重协调了才是最合适的。工作中写过的算法，有考虑过它的时间消耗？写出来个递规了就感觉牛逼了，有想过这种算法的空间开销？
