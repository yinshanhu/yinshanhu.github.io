    # 算法之冒泡排序

    2019.03.11

    冒泡排序是编程中最简单的一种算法了，也是一种入门级的算法。冒泡排序的算法时间平均复杂度为O(n²)。空间复杂度为 O(1)。冒泡排序为稳定排序。

    # 执行过程

    步骤：

    ①程序进行n-1趟比较（设有n个数）。

    ②每趟两两比较，若前者大于后者，则相互交换位置，那么该趟执行结束，最大数置于该趟的最后一位，在下一趟该趟的最后一位数不参与比较。

    ③如此n趟执行结束，便得从小到大排序。

    若是从大到小排序，在第②步中，若前者小于后者，则交换位置即可

    # javascript代码实现

    function bubbleSort(arr) {
    let n = arr.length;
    for(let i = 0; i < n-1; i++) {
        for(let j = 0; j < n-i-1; j++) {
            if(arr[j] > arr[j+1]) {                
                //交换位置
                [arr[j], arr[j+1]] = [arr[j+1], arr[j]];

                //let tmp = arr[j];
                //arr[j] = arr[j+1];
                //arr[j+1] = tmp;
            }
        }
    }
}

    # 复杂度分析

    • 时间复杂度：O(n²)

    长度为n的数组，最外层for要进行n-1趟循环，每趟要进行n-i-1次执行，

    即：第0趟执行 n-1次，第1趟执行n-2次 ...... 最后一趟执行1次，执行次数相加，

    即：(n-1)+(n-2)+......+1 = n * (n - 1) / 2

    O(n²)表示的是复杂度的数量级，其它系数可以忽略。

    • 空间复杂度：O(1)

    算法所需空间资源没有随程序执行次数增加而增大。

    # 是一种稳定排序算法

    • 排序算法的稳定性：通俗地讲就是能保证排序前两个相等的数据其在序列中的先后位置顺序与排序后它们两个先后位置顺序相同。

    # 总结

    1. 冒泡排序的算法时间平均复杂度为O(n²)。
    2. 空间复杂度为 O(1)。
    3. 冒泡排序为稳定排序。
